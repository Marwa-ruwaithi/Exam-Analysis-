# Exam Time vs Performance Analysis

This project analyzes the relationship between the estimated time allocated for each exam question part and student performance.
The analysis compares male and female sections using a cleaned Excel dataset and simple Python analysis.

## Requirements
Before running the project, make sure you have:
- Python 3
- The following Python libraries:
  - pandas
  - matplotlib
  - scipy
  - openpyxl

You can install the required libraries using:

pip install pandas matplotlib scipy openpyxl
